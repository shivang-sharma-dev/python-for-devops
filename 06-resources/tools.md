# Tools

### Must-have packages
| Package | Install | What it does |
|---|---|---|
| `requests` | `pip install requests` | HTTP requests |
| `boto3` | `pip install boto3` | AWS SDK |
| `pyyaml` | `pip install pyyaml` | YAML parsing |
| `click` | `pip install click` | CLI tools |
| `rich` | `pip install rich` | Beautiful terminal output |
| `python-dotenv` | `pip install python-dotenv` | Load .env files |
| `paramiko` | `pip install paramiko` | SSH from Python |

### Development tools
| Tool | Install | What it does |
|---|---|---|
| `black` | `pip install black` | Auto-format your code |
| `flake8` | `pip install flake8` | Lint for style errors |
| `mypy` | `pip install mypy` | Type checking |
| `pytest` | `pip install pytest` | Run tests |
| `ipython` | `pip install ipython` | Better interactive shell |
